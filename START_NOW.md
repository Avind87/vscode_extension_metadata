# Start the Extension NOW

## Quick Steps

1. **Make sure this folder is open in Cursor:**
   `/Users/kjemac/GIT/metadata_frontend`

2. **Press F5** (or Cmd+Shift+D then click "Run Extension")

3. **In the new window that opens:**
   - Press `Cmd+Shift+P` (macOS) or `Ctrl+Shift+P` (Windows/Linux)
   - Type: "Open TurboVault Metadata Editor"
   - Select it
   - Choose `test_metadata.duckdb` file

That's it! The extension should launch.

## Alternative: Using Terminal

If F5 doesn't work, you can try opening the debug panel:
- Press `Cmd+Shift+D` (or View → Run and Debug)
- Select "Run Extension" from dropdown
- Click the green play button ▶️
